  <name>Task 1: Implement ResidualMAPPOTrainer with act_batch_residual and joint logp update</name>
  <files>ece324_tango/asce/mappo.py</files>
  <behavior>
    act_batch_residual returns per-agent dicts with: "action" (final dispatch), "logp" (joint logp),
    "value" (critic), "gate" (0 or 1), "mp_action" (passed-in mp_actions_list value)

    joint logp invariants:
    - gate=0: logp_joint = log P(gate=0); phase logp NOT added
    - gate=1: logp_joint = log P(gate=1) + log P(sampled_phase | gate=1)

    action dispatch invariants:
    - gate=0: final_action = mp_actions_list[i]
    - gate=1: final_action = sampled phase from phase_head

    PPO update joint logp recomputation:
    - For gate=0 transitions: new_logp = gate_dist.log_prob(0); phase head contributes 0 gradient
    - For gate=1 transitions: new_logp = gate_dist.log_prob(1) + phase_dist.log_prob(phase_action)
    - Implementation: use gate_mask = (gates_tensor == 1).float()
      new_logp = gate_logp + gate_mask * phase_logp
      This means gate_mask=0 multiplies phase_logp by 0 → zero contribution → zero gradient through phase_head

    build_batch must include mp_action and gate arrays from Transition fields

    save/load: when residual_mode=action_gate, payload includes "actor_gate" (GatedActor state_dict);
    when loading, validates residual_mode matches checkpoint metadata

    residual_mode="none": ResidualMAPPOTrainer behaves exactly like MAPPOTrainer
    (use plain Actor, act_batch unchanged, update unchanged)
  </behavior>
  <action>
    Add ResidualMAPPOTrainer class to mappo.py after MAPPOTrainer. It inherits from MAPPOTrainer for backward compat.

    Key implementation notes:
    1. Constructor: accept residual_mode="none"|"action_gate" and gate_init_bias=-2.0.
       When residual_mode="action_gate": self.actor = GatedActor(obs_dim + n_actions, n_actions, hidden_dim, gate_init_bias)
       (obs_dim is already augmented — the augmented obs goes into the actor)
       When residual_mode="none": self.actor = Actor(obs_dim, n_actions, hidden_dim) (identical to MAPPOTrainer)

    2. act_batch_residual(obs_list, global_obs, n_valid_actions_list, mp_actions_list):
       - Augment obs: augmented = augment_obs_with_mp(np.stack(obs_list), mp_actions_list, self.n_actions)
       - Normalize augmented obs if obs_norm is set
       - Forward GatedActor: gate_logits [N,2], phase_logits [N, n_actions]
       - Mask invalid phase actions (same as existing act_batch masking)
       - Sample gate (Categorical over gate_logits)
       - Sample phase (Categorical over masked phase_logits)
       - Compute joint logp: gate_logp + (gate==1).float() * phase_logp
       - final_action[i] = mp_actions_list[i] if gate[i]==0 else phase_action[i]
       - Return list of dicts: {"action": final_action, "logp": joint_logp, "value": value,
                                "gate": gate, "mp_action": mp_actions_list[i]}

    3. build_batch: override to include "mp_action" and "gate" arrays from Transition.mp_action, Transition.gate

    4. update: override to use joint logp recomputation.
       - Extra tensors loaded from batch: gates = batch["gate"], mp_actions = batch["mp_action"]
       - In PPO minibatch loop, after computing new logits:
           gate_logits_mb, phase_logits_mb = self.actor(obs[mb])  # GatedActor returns tuple
           gate_dist = Categorical(logits=gate_logits_mb)
           gate_logp_mb = gate_dist.log_prob(gates[mb])
           # Apply action mask to phase logits
           phase_logits_mb = <mask invalid actions same as current code>
           phase_dist = Categorical(logits=phase_logits_mb)
           # Only compute phase logp for gate=1 transitions
           phase_actions_for_logp = actions[mb]  # stored final_action; gate=0 rows use mp_action
           # For gate=0 rows, phase logp is 0 (multiplied by gate_mask)
           gate_mask = (gates[mb] == 1).float()
           phase_logp_mb = phase_dist.log_prob(phase_actions_for_logp) * gate_mask
           new_logp = gate_logp_mb + phase_logp_mb
           # PPO ratio uses new_logp vs old_logp (stored joint logp)
           ratio = torch.exp(new_logp - old_logp[mb])

       CRITICAL: For gate=0 transitions, gate_mask=0 ensures phase_logp_mb=0,
       which means no gradient flows through phase_head for those transitions.
       Verify this is exactly what test_gate0_phase_head_zero_gradient tests.

    5. save: when residual_mode=action_gate, add "residual_mode": "action_gate" and
       "actor_gate": self.actor.state_dict() to payload.
       When residual_mode=none: payload identical to MAPPOTrainer.save().

    6. load: check payload["residual_mode"] matches self.residual_mode.
       Raise RuntimeError with clear message if mismatch.

    Phase actions for gate=0 transitions in PPO update:
    The stored Transition.action is final_action (= mp_action when gate=0).
    When recomputing phase_logp for gate=0 rows, we use those mp_action values, but
    multiply by 0 (gate_mask=0), so no gradient flows through phase_head regardless.
    This is the correct implementation — do not substitute a different action value.
